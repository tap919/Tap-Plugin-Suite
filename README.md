# Tap-Plugin-Suite
Windows-only VST3 plugin suite built with JUCE.

## Suite specification
The full JSON scaffold for the suite lives in the `Tap Plugins` file (no extension) at the repo root.

## Readiness (75–100%)
The readiness checklist for each plugin is tracked in `docs/readiness.md`.

## Scaffolding code
Early DSP scaffolding lives in `src/` as standalone C++ processors for each plugin.
